## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

* Expo-CLI
  ```sh
  npm install --global expo-cli
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/KPT1020/workout-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run the app
   ```sh
   expo start
   ```
